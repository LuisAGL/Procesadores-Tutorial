# Seguidor de luz aplicable a paneles solares

## Componentes
- Arduino uno
- 2 servomotores
- 4 Fotoresistencias
- 4 resistencias
- Jumpers
- 4 Resistencias

## Funcion
Buscar y seguir la mayor fuente de luz encontrada para dar mejor regimen de energia.
